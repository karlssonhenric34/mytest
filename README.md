##### 京东隐私安全 环境变量

|      Name       |    归属     |  属性  | 默认值 | 说明                                                         |
| :-------------: | :---------: | :----: | :----: | ------------------------------------------------------------ |
|   `JD_DEBUG`    | 脚本打印log | 非必须 |  true  | 运行脚本时，是否显示log,默认显示。改成false表示不显示，注重隐私的人可以设置 JD_DEBUG 为false |
| `JD_USER_AGENT` |    京东     | 非必须 |        | 自定义此库里京东系列脚本的UserAgent，不懂不知不会UserAgent的请不要随意填写内容。如需使用此功能建议填写京东APP的UA |
